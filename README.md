## Cómo usar este repo: 💻 


Si querés probar este repo, clonalo usando
```bash
git clone https://github.com/Maoi11/tpReact-ejercicio2.git
```
Y luego ejecutá los siguientes comandos: 
- npm install
- npm run dev
